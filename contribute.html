Guidelines about contributing content.

 

1. Only share content that you made.

2. Must answer responses in the pre-filled email.

3. If contributing a video, you must send a link to the video (preferably an unlisted Youtube link).

4. If contributing a worksheet and its answer key, you must send a link to the document (preferably a Google Drive link).

5. Content may be removed from the content vault at any time, if there is a senseless amount of data in a single topic.

6. If you have any questions, look in the help page of the site.

 

Thank you for taking the time to help out others!
